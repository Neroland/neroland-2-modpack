# Neroland 2

The official repository and issue tracker for **Neroland 2** (Minecraft 26.2,
NeoForge).

## Reporting issues

Please use the issue templates. Include your pack version, a crash log (via
[mclo.gs](https://mclo.gs)), and clear reproduction steps.
